# Agent Notes

Canonical instructions for coding agents live in `AGENTS.md`. This lowercase file is kept as a discoverable pointer for tools or editors that look for `agent.md`.

Read `AGENTS.md` first, then load the matching project skill from `.agents/skills/`:

- `wc3-ui-designer` for general Electron/TypeScript editor work.
- `wc3-undo-commands` for adding or changing undoable/redoable frame command classes.
- `wc3-export-safety` for generated JASS, Lua, TypeScript, TOC, and FDF changes.
- `graphify` for codebase graph queries and graph maintenance.
